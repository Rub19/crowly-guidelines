Crowly – Bot Guidelines

Description : Crowly est un bot multifonction conçu pour la modération, les giveaways et les statistiques sur Discord.

1. Utilisation

  • Le bot doit être utilisé uniquement sur des serveurs respectant les règles d’utilisation de Discord.
  • Aucune utilisation malveillante (spam, raids, insultes, etc.) n’est autorisée.
  • L’équipe de développement se réserve le droit de bloquer l’accès au bot en cas d’abus.

2. Collecte de données

  • Crowly ne collecte que les informations nécessaires à son fonctionnement (ID d’utilisateur, nom de serveur, etc.).
  • Aucune donnée n’est partagée ni vendue à des tiers.
  • Les données sont utilisées uniquement pour améliorer l’expérience utilisateur.

3. Support

Pour toute question ou problème, contactez Rub19 sur Discord.

4. Sécurité

  • Crowly respecte les conditions d’utilisation de l’API Discord.
  • Aucune action dangereuse ou non autorisée n’est exécutée sans consentement.

5. Confidentialité

La confidentialité des utilisateurs est une priorité. Aucune donnée personnelle n’est enregistrée durablement.
